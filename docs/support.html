Questions fréquentes

Comment réinitialiser mon compteur ?

Dans l'app, va dans Réglages → Ma cigarette de départ et modifie la date de ta dernière cigarette. Le compteur se recalcule automatiquement.

J'ai rechuté — que faire ?

Sur l'écran d'accueil, appuie sur le bouton de rechute. Tu peux réinitialiser ton compteur à maintenant. Une rechute ne remet pas à zéro tes badges ni ton journal.

Mes données sont-elles sauvegardées ?

Par défaut, tout est stocké localement sur ton appareil. Si tu crées un compte Respire dans Réglages → Compte, tes données sont synchronisées dans le cloud et accessibles sur plusieurs appareils.

Comment activer les notifications ?

Dans Réglages iOS → Respire → Notifications, active les autorisations. Puis dans l'app, configure l'heure de rappel dans Réglages → Notifications.

J'ai acheté Premium mais il n'est pas activé

Va dans Réglages → Restaurer les achats. Si le problème persiste, contacte-nous avec le reçu Apple à antoinequarroz.mediam@gmail.com.

Comment supprimer mon compte et mes données ?

Dans l'app : Réglages → Compte → Supprimer mon compte. Toutes tes données sont effacées définitivement de nos serveurs dans les 30 jours.

Le widget ne s'affiche pas

Appuie longuement sur l'écran d'accueil → "+" → cherche "Respire". Si le widget n'apparaît pas, désinstalle et réinstalle l'app, puis ouvre-la au moins une fois avant d'ajouter le widget.
